## 训练方案

由于标签文件不完整，我提供了以下解决方案：

### 方案1: 迁移学习训练 (推荐)
使用预训练模型进行微调，即使标签很少也能利用预训练知识。

**使用方法:**
1. 双击运行 `train_apple_now.bat`
2. 按照提示选择选项1 (迁移学习训练)
3. 等待训练完成 (约15-45分钟)

**训练参数:**
- 训练轮数: 30轮
- 批次大小: 4
- 图像尺寸: 416x416
- 学习率: 0.001 (微调)
- 冻结层数: 10层

### 方案2: 直接使用预训练模型
YOLOv8n模型已经具备通用物体检测能力，可以直接使用。

**使用方法:**
1. 确保 `fast_camera_app.py` 中使用 `yolov8n.pt`
2. 直接运行应用

## 提高精度的建议

### 短期方案
1. **使用迁移学习** - 利用预训练模型的知识
2. **数据增强** - 增加训练数据的多样性
3. **调整参数** - 优化学习率、批次大小等

### 长期方案
1. **补充标注** - 为所有图片添加准确标签
2. **收集更多数据** - 增加苹果图片的数量和多样性
3. **使用更大模型** - 如yolov8s.pt或yolov8m.pt

## 标注工具推荐

如果你需要标注图片，推荐以下工具：

### 免费工具
1. **LabelImg** - 简单易用的标注工具
2. **CVAT** - 功能强大的在线标注平台
3. **Roboflow** - 在线数据集管理平台

### 标注格式
YOLO格式的标签文件应为 `.txt` 文件，每行格式：
```
class_id x_center y_center width height
```

示例：
```
0 0.5 0.5 0.3 0.4
```

## 常见问题

### Q1: 训练需要多长时间？
A: 在CPU上约15-45分钟，取决于电脑性能。

### Q2: 训练需要多少内存？
A: 约2-4GB内存，批次大小设置为4以减少内存使用。

### Q3: 训练后模型保存在哪里？
A: 训练结果保存在 `runs/detect/apple_transfer_learning/` 目录，最佳模型会复制到 `apple_transfer_best.pt`。

### Q4: 如何知道训练是否成功？
A: 训练完成后会显示训练曲线和模型文件路径，可以使用测试脚本验证模型效果。
